REACT NATIVE DEMO

REQUIRED INSTALLATIONS
1. Java jdk
2. Android studio with necessary updates, API and SDK
3. Node.js
4. install the react-native-cli globally

ENVIRONMENT VARIABLE SETUP
1. Add ANDROID_HOME environment variable with address of the android sdk
2. Add the same address to the path environment variable
3. Add java jdk to path environment variable

PROJECT SETUP
1. Download project folder as zip
2. extract contents to desktop
3. open powershell
4. navigate to the project folder
5. run the command 'npm install'
6. Open an android emulator from the command line or through android studio
7. run the command 'react-native run-android' (project should open in the emulator automatically)
